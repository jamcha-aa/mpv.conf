# mpv.conf
Author: jamcha (jamcha.aa@gmail.com)

This mpv.conf targets cheap APU machine without proprietary graphic driver (e.g., RADEON HD 6310).

## Usage

Install mpv.

Put mpv.conf to ~/.config/mpv/

Enjoy.
